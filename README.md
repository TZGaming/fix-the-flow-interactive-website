# Embassy of the Free Mind
Dit is mijn website voor Embassy of the Free Mind. Dit museum wilt graag hun boeken digitaal beschikbaar hebben, zodat je ze ook op hun eigen website kan bekijken.

Ik heb gekozen voor een donker & licht contrast. Zelf heeft het museum niet echt een uitgebreid kleurspectrum. Hun kleur keuzes bestaan uit donker blauw, geel/goud en wit, alle 3 de kleuren heb ik op verschillende manieren toegepast. Ik heb goud gekozen voor de links in de header en footer, want goud als background of header/footer kleur werkt gewoon niet goed qua contrast.

### Javascript interactie
Ik heb ook een interactie gemaakt in JavaScript, namelijk voor dark/light mode. Ik vind het belangrijk dat mensen de website in het donker kunnen zien, want niet iedereen houdt van fel licht. In de header zie je een maan en zon, als je daarop klikt dan veranderd of add hij styles in de HTML die aparte kleur configuraties hebben.

Ook heb ik een Nederlandse en Engelse flag toegevoegd, zodat je de taal van de website kan veranderen. Dit is voor het geval dat iemand niet Nederlands begrijpt.

## Kenmerken
De CSS heeft een goede vaste structuur, met alles goed op volgorde van hoe het in de HTML staat. Dit was 1 van de dingen die ik nog graag wou toepassen op mijn re-design. Ook heb ik elke pagina een aparte CSS gegeven, omdat sommige paginas andere configuraties nodig hebben voor bepaalde elementen. Anders moet ik heel veel classes aanmaken en word de code echt een mess.
